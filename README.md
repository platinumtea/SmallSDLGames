# SmallSDLGames
A group of smaller games made in SDL.

## [Tic-Tac-Toe](https://github.com/platinumtea/SmallSDLGames/tree/master/TicTacToe)
A hotseat version of Tic-Tac-Toe, click to draw symbol. Play starts with X and alternates.
